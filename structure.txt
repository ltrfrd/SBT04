SBT04
School bus operations backend with route/run management, flexible runtime stop execution,
pre-trip safety enforcement, route bus control, planned run scheduling, and dispatch alerts.

===============================================================================
IMPLEMENTED SYSTEM NOTES
===============================================================================

- One pre-trip exists per bus per day
- Pre-trip input is user-facing through bus_number and license_plate
- Internal bus lookup still uses Bus.unit_number and internal relationships still use bus_id
- Active bus is authoritative for run-start safety checks
- assign_bus sets the active bus and seeds the primary bus when empty
- restore_primary_bus moves the active bus back to the primary bus
- Runs store scheduled_start_time and scheduled_end_time in addition to actual start_time and end_time
- Run start is blocked when no valid pre-trip exists for today's active bus, when fit_for_duty is no, or when any major defect exists
- DispatchAlert stores major-defect, not-fit, and missing-pretrip alert records
- Runtime stop execution remains flexible through arrive_stop, next_stop, pickup, and dropoff endpoints
